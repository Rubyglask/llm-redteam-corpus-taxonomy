# 2026-Q3 Quarterly Report

> **Status:** v0.1.0 datasets live on Hugging Face (measured 2026-07-03).

## What shipped

- **Datasets published** — `core` (44,681 prompts, CC-BY-4.0) and `extended`
  (26,748 prompts, CC-BY-NC-SA-4.0) on Hugging Face, loadable via
  `load_dataset`. 12 sources shipped (9 core + 3 extended); 3 excluded on
  purpose (StrongREJECT / WMDP / Persuasion), 2 pending license check.
- **Reproducible pipeline** — deterministic build + SHA-256 checksums; PII
  masking; per-source `sources.json` attribution.
- **Taxonomy v0.1** — OWASP LLM Top 10 anchor + confidence-tagged crosswalk
  (framework-to-framework crosswalks already exist publicly; ours is a
  convenience copy for use with the corpus).
- **Evidence timeline** — weekly HF metrics tracking started (baseline below).
